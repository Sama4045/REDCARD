# REDCARD Project

This project is a simple multi-page website created using **HTML** and **CSS**.

## 📌 Project Description

The project contains two web pages:

- **index.html** – Main page
- **screen2.html** – Second page

Both pages use a shared CSS file (`style.css`) for styling and an `assets` folder for images.

## 🛠️ Technologies Used

- HTML5
- CSS3

## 🚀 How to Run the Project

1. Download or clone this repository.
2. Open the project folder.
3. Open `index.html` in your web browser.
4. Navigate to the second page using the provided link or button.

All images used in this project are stored inside the `assets` folder.
